Движение

обратная связь, не понты

ЗагрузкаСкелетоны вместо крутилок + ступенчатое появление карточек. Ощущение скорости.
ДействиеТост-подтверждение на каждое действие. Числа «накручиваются», кнопки реагируют мгновенно.
ЭффектыGlare, стекло, наклон — только на лендинге и AI-моментах. В кабинете — спокойствие.
SERA · Design System — на реальных токенах src/styles/tokens.css. Бренд = тема поверх shadcn. Все страницы равняются сюда.
