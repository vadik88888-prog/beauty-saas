## 🔵 РЕШЕНО ПО ХОДУ

- **AI работает на OpenAI (GPT), не на OpenRouter.** Остаёмся на GPT — оплата API из РБ решаема через сервисы-пополнялки; OpenRouter из РБ оплатить сложно.
- **Селектор модели в настройках SERA — РАБОТАЕТ** (не фейк). Выбранная модель реально уходит в запрос (читается из `tenant_ai_settings.model`). По умолчанию `gpt-4o-mini` — только запасной вариант.
- **GPT-5.2 примерно в 40× дороже, чем gpt-4o-mini.** Качество выше (важно для записи с инструментами), но это прямая себестоимость — учитывать в цене подписки.
- Два служебных вызова (кнопки быстрого ответа + сжатие диалогов) жёстко на `gpt-4o-mini` — это правильно, не трогать.

## 🌸 КАК ВЫГЛЯДИТ ГОТОВЫЙ ПРОДУКT (простыми словами)

Представь два входа в одно здание.

**Вход первый — для владельца салона (SaaS-панель).**
Это «пульт управления» салоном. Тёмно-зелёное меню слева с живым огоньком SERA. Владелец заходит и за минуту видит: сколько сегодня записей, кто придёт, сколько заработал, кто из клиентов давно не появлялся. Все страницы — в одном спокойном кремовом стиле, без «фальшивых окон» (каждая цифра реально посчитана). Внутри везде помощник SERA: подсказывает, где пустые окна, кого вернуть, какую акцию запустить. Владелец не нанимает администратора — его работу делает SERA, а владелец только подтверждает важное.

**Вход второй — для клиента салона (Telegram Mini App).**
Клиент открывает SERA в Telegram как обычное приложение. Видит услуги, мастеров, свободное время — записывается в пару нажатий. Может просто написать SERA словами «хочу маникюр в пятницу вечером» — и она сама подберёт время, запишет, напомнит за день, поздравит с днём рождения, позовёт обратно, если давно не был. Для клиента это «идеальный администратор, который всегда онлайн и никогда не забывает».

**Главный принцип — честность и доверие.**
SERA сама делает безопасное (напомнить, подобрать слот, ответить на вопрос). Но всё рискованное (отправить клиенту сообщение, отменить запись) — **только с подтверждения человека**. Поэтому салон ей доверяет. Доверие — это то, за что платят.

**Куда всё растёт (финал).**
Когда салонов на платформе станет много — открывается третий слой: клиент открывает SERA, она видит его город, и показывает **все салоны рядом**, работающие на SERA. Клиент выбирает и записывается прямо там. SERA становится не просто пультом для одного салона, а **единой картой бьюти-города**: где запись, общение, маркетинг и AI живут в одном месте. Это и есть цель — стать стандартом, как салоны вообще управляются.

**Одной фразой:** SERA — это идеальный AI-администратор внутри Telegram для клиента + пульт управления с AI-помощником для владельца, которые со временем срастаются в общую сеть салонов и клиентов.
